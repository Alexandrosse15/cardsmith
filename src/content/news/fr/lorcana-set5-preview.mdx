Shimmering Skies, le cinquieme set de Disney Lorcana, arrive en juin. Les previews tombent et certaines cartes font deja parler d'elles sur les forums.

## Les cartes qui retiennent l'attention

**Stitch - New Dog** ressort dans une version 2 encre avec une capacite qui permet de piocher quand un de vos personnages est defie. Dans la meta actuelle centree sur les defis, ca peut devenir un moteur de main consistant.

**Maleficent - Biding Her Time** revient avec une version Legendaire qui impose une penalite aux adversaires qui jouent des chansons. Potentiellement tres forte contre les decks Amber actuels.

## Impact sur les prix

Les cartes previews genèrent toujours de la speculation. Sur Cardmarket, les precommandes pour les Legendaires deja reveles gravitent autour de 20-35 euros, soit dans la fourchette habituelle pour un set non sorti.

Notre conseil : attendez la release complete avant d'acheter. Les prix des Legendaires chutent generalement de 30 a 40 % dans les deux semaines qui suivent la sortie, a mesure que le marche se stabilise.

## Ce qu'on attend encore

Les previews de la semaine prochaine devraient couvrir les nouvelles cartes Sapphire, la couleur qui manque encore dans les revelations. Revenez sur Cardsmith pour les analyses au fil des previews.
